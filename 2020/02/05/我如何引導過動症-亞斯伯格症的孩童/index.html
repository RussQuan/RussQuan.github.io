我如何引導過動症+亞斯伯格症的孩童?

Posted on 2020-02-05 | 0 comments

題外話

我不是專業醫師或是專業的輔導師，只是23歲的電子系學生。
其實在發文之前，我思考了許久到底要不要分享我的一些想法跟觀念，因為畢竟我不是專業的輔導師，深怕自己撰寫的文章可能會有錯誤的觀念去誤導大家。
不過我很開心可以在留言區看到大家給我的鼓勵跟討論，並且期待我可以再分享更多的文章
有時間我會繼續分享與孩子相處的經驗
謝謝大家，我們一起加油

前言

我的第一篇文章《我如何引導過動症+亞斯伯格症的孩童?》，跟大家聊到這些孩子們的情緒迴圈以及如何透過引導的方式讓孩子透過正向的思考解決問題。
這次想用我3年前在補習班打工時，與每位國小國中孩子們談心的經驗，一樣用同理心的出發點，讓大家理解為什麼孩子不想寫作業，希望可以幫助到被相同問題困擾著的家長們。

孩子完成了什麼?

原本狀態
家長怎麼喊都喊不動，孩子拖延、吵鬧、逃避。

現在狀態

  1. 主動拿作業到位子上寫
  2. 主動拿作業請教老師
  3. 甚至題目只要求寫上3個英文單字，孩子卻自己主動要求想寫到5個甚至6個 (因為想學習而主動學習)

我家的孩子為什麼不主動寫作業?

其實不然是孩子，當家長在工作岡位上首次遇見難題時，可能也出現過

  • 為什麼老闆要叫我做我不擅長的事情 (抱怨)
  • 這個問題搞得我心煩氣躁，明天再處理好了 (拖延)
  • 還是辭職算了… (逃避)

為什麼會出現這些反應呢?
因為

  • 首次接觸這個問題，不知道用什麼方法來解決
  • 問題與目前自身程度相差甚遠，不知從何起頭
  • 身旁沒有人可以指導

總而言之，就是 ==我不知道該怎麼做==

別再叫孩子去寫作業了

您看到這裡可能會覺得疑惑，不是希望看到孩子完成作業嗎? 怎麼又說叫他們別寫了。
前面我用家長工作可能遇到的情況，去比喻孩子之所以不想寫作業的原因。

當出現這個情況時，就可能負面的心理迴圈，想著 ==我怎麼什麼都不會== 的思緒

這個時候家長若叫孩子 ==趕快去寫作業== ，就相同於老闆與您說，==這份報表何時能交出?== 是一樣的道理。

當你不知道這份工作、功課、問題，該如何解決時，其實自己給自己的壓力就已經很重了，這個時候上級、家長、身旁的人，不但沒有幫你分析問題、提出有效的解決方法，甚至施加壓力，這個時候情緒低迷的你，要怎麼充滿幹勁的完成預達成的目標呢?

而您是不是也將同樣的壓力，在無形之中施加到孩子的身上了呢?

到這裡我只點出我認為孩子對寫作業提不起勁、排斥寫作業的原因
接下來我要告訴你，如何實際改善這些狀況，讓孩子漸漸愛上寫作業，最終達成主動寫作業的目標

引導孩子完成目標 (以完成寒假作業為例)

許多家長可能會用==買喜歡的東西作為誘導==，或是用嚴厲斥責讓孩子乖乖地坐上工作區。

==這些方法往往會產生負面的效果==，至於為什麼，如果大家想知道，我可能在花一篇文章的時間跟大家解釋我在這個部分的觀點。

接下來我用實際的例子，帶你了解怎麼引導孩子完成作業，或是任何的目標

例子1

現在正值寒假，如果以==完成 “寒假作業” 當作目標==，我該怎麼做?

家長作法 → 快去寫功課
我的作法 → 我們今天不寫功課，在寫功課前，我們要先做更重要的事情

教他

  • 時間管理 (讓他知道寒假剩餘多少時間，功課量有多少，並安排自己的計劃)
  • 拆解並區分作業項目 (把要完成的寒假作業從最簡單到最難區分，並把無法獨力完成需要請教他人的作業圈選出來)
  • 實際的帶孩子操作這些方法一遍

讓我們實際打開寒假作業

  1. 打開寒假作業目錄
  2. 讓題目變的白話文
    • 例如 幸福的私房菜 根據題目把他拆解為幾個步驟
      • 做一道佳餚
      • 拍照、列印
      • 寫上心得
  3. 依照孩子認為的難易度，從簡單到難排序
  4. 需要別人幫忙的作業，打三角形

其他作業的概略方式大同小異，但根據不同科目可能有些許差異

以身作則

家長在請孩子去寫作業的同時，是不是自己正在滑手機、看電視，做著孩子眼中是娛樂的事情呢，上一篇文章提到以身作則的重要性，請您暫時先放下手邊的工作或是娛樂，花些時間來了解孩子在課業的哪裡卡住了、停頓了，是因為專注力不夠了，還是因為這個問題的解法想不出來。

這樣子除了可以增加孩子與您的相處時間，更能培養孩子對您的信任感，讓孩子知道當孩子遇到困難及挫折的時候，您會引導他，而不是斥責或是請他自己完成。

結論

  1. 將施壓在孩子身上的壓力收回
  2. 父母以身作則
  3. 幫助孩子拆解題目
  4. 讓他從中獲得成就感
